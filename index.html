How to Use

To get Started, You Have To Have The Base URL For The API, For our Hosted Server:
http://35.209.150.159/

Then You Would Have to Follow The Instructions Below For How to Use Every Endpoint with A Correct Parameters

Responses Strucure

For Every Endpoint, the Response Will Be Consist of:

success With Values true/false:

If the Request is Done Correctly Without Error, it is True, otherise it is False

data:

if the Request Successed, if it should Return a Data, it will be Name "data"

message:

When the Server Wanted to Tell Somthing to The Sender

For Example, To Use URL to Login, The URL Will Be Like:
http://35.209.150.159/login/
